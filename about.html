About this project

Hey, my name is Max, if you are anything like me, you find standard font in the HSK examination system impossible to read. My solution to this problem is simlple: train reading such text to get used to it and not feel frustraited during the exam. This system provides several texts for part 3 （写作） with this HSK-like shitty font and timers so you can practice at home. Note, that some texts are generated with AI and are not actual legit traning material, but I feel like they are close enough. Good luck practicing.
